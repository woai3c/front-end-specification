### 在线阅读
* [Github 访问地址](https://woai3c.github.io/front-end-specification/)

如果在线访问比较慢，建议你克隆项目在本地观看，下载项目后执行以下命令：
```bash
# 安装依赖
npm i
# 启动服务器
npm run dev
```
然后在浏览器打开 `http://localhost:8080` 进行访问。
